Credit card fraud costs financial institutions billions annually. This project demonstrates an end-to-end machine learning pipeline to detect fraudulent transactions using the **ULB Credit Card Fraud Dataset**. 

We tackle the extreme class imbalance (0.17% fraud) using **SMOTE** and compare two distinct architectures:
1. **Random Forest:** An ensemble method for robust tabular classification.
2. **Deep Neural Network (MLP):** A Keras-based model to capture complex non-linear patterns.
